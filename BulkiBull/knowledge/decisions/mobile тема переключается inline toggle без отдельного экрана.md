# Mobile тема переключается inline toggle без отдельного экрана

Для текущего MVP оформление в mobile не требует отдельного экрана настроек. Пользователю достаточно быстрого переключателя прямо из верхней панели, чтобы мгновенно проверить светлую или темную тему.

## Почему так

- отдельный экран ради одной настройки перегружает навигацию;
- переключение темы это частый визуальный toggle, а не отдельный workflow;
- inline-кнопка проще тестируется и быстрее проверяется в iOS Simulator;
- так легче сохранить основной продуктовый фокус MVP на бычках, весе и кормах, а не на разделе настроек.

## Как это устроено

- тема управляется централизованно через `apps/mobile/src/theme/theme.ts`;
- header использует компактную кнопку-тоггл вместо перехода на отдельный route;
- визуальная часть темы опирается на Ionic dark palette и app-level CSS variables.
