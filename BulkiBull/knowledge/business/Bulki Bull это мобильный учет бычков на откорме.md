# Bulki Bull это мобильный учет бычков на откорме

Bulki Bull - MVP мобильного приложения для учета бычков на откорме.

## Базовая продуктовая идея

Пользователь должен вести понятный мобильный учет животных: список бычков, карточка животного, записи веса, история изменений и позже фото. Текущий README фиксирует, что бизнес-сущности, CRUD и мобильные MVP-сценарии пока намеренно не реализованы.

## Ближайшие сущности

- `Bull` - животное в учете.
- `WeightRecord` - запись веса по конкретному животному.

Эти модели запланированы на Stage 2. См. [[stage 1 оставляет бизнес-модели Prisma на следующий этап]].

## UX-направление

Мобильный интерфейс должен быть быстрым и полевым: минимум лишнего текста, крупные действия, быстрый доступ к списку, карточке, добавлению веса и просмотру истории.
